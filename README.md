# Refnd
Toolkit to handle datasets generated from a Relational Generative Process (RGP). It is implemented in Rust for speed and efficiency, and contains python bindings.

## Installation – Python
Currently, we can only install from source. Run:
```shell
pip install "git+https://github.com/anthol42/refnd.git#subdirectory=py"
```

Pre-compiled releases to facilitate installation are comming soon!
