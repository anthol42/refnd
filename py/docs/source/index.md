# py-proto

**py-proto** is a Python binding for the `proto-core` Rust crate. It provides fast,
SIMD-accelerated tools for working with datasets generated by a *Random Generative
Process* (RGP) — datasets where items share latent structure (e.g. protein families, molecules)
that makes naive random splits leak information between train and test.
