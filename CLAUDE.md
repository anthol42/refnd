## Mindset

**Short and sweet.**
- *Short*: concise code, no repetition, no boilerplate, no speculative abstractions.
- *Sweet*: readable and maintainable — the next reader should understand intent immediately.

Write like a software engineer: correct, efficient, well-structured. But never write more than the task requires.

---

## What this is 
This project implements a rust crate to handle dataset generated by a RGP. A RGP is a two step 
generative process that will generate data groups that are related - or dependent - on each other. 
Similar to how similar proteins have a similar function, or similar organic molecules get similar 
properties. Data originating from RGP are problematics for dataset split as we cannot simply do a 
random split as it will lead to data leakage. 

Here, we implement a rust toolkit designed to make it easier to work with RGP datasets. It containes the main crate 
under `proto-core` and a python binding under `pyproto`.
